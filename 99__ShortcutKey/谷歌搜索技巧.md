##  使用引号强制进行精确匹配搜索

```
"what is javascript"
```



## And 运算符

`And` 运算符只会返回和它们都相关的搜索结果

```
html AND css
```



### 使用 OR 操作符获取搜索词中某一个的结果

```
(javascript OR python) free course
```



### - 操作符将排除包含搜索词的结果

```
javascript -css
```



### 可以使用(*)通配符作为占位符，它将被任何单词或短语替换

```
"how to start * in 6 months"
```



### 在一个单一的网站搜索

```
site:freecodecamp.org
```



### 查找特定的文件类型

```
filetype:pdf learn css
```



###  搜索一个数字范围

```
ecmascript 2016..2018
```
